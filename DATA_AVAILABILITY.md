# Data Availability

This repository contains the code, tests, TeX sources, manuscript PDFs, derived
tables, derived figures, and small derived CSV artifacts used for the MESA
working paper.

Raw market-data files are not included. The LOBSTER inputs used by the scripts
are public sample or licensed files subject to LOBSTER terms, and Binance
aggregate-trade archives can be fetched from public sources using the included
scripts. The repository includes only derived outputs needed to inspect and
reproduce the reported figures and tables.

To regenerate the public-data inputs where allowed, see `README_REPRODUCE.md`
and the scripts under `scripts/`.
